# Focus Track

Window activity tracker for Linux

Tracks your activity accross the open application and show how much time was spent in each of them.
Can group activities by app and by window.

## DYI - you can build this app yourself with your favorite LLM
Here in the repo there is a file with the prompt PROMPT.md

You can put this prompt into your favorite LLM (I tested with Claude Sonnet 4.6) and it will create the app for you.
